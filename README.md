# 🚀 Nilabh Deka - 3D Portfolio

A stunning 3D portfolio inspired by Warhammer 40k, built with Three.js and modern web technologies. This portfolio showcases Nilabh's journey as a techno-architect specializing in backend development, machine learning, and system architecture.

## ✨ Features

- **Immersive 3D Environment**: Warhammer 40k-inspired futuristic cityscape
- **Particle Systems**: Dynamic energy particles, smoke effects, and atmospheric elements
- **Smooth Animations**: Fluid transitions and interactive 3D elements
- **Responsive Design**: Optimized for all devices and screen sizes
- **Audio Integration**: Atmospheric sound effects for enhanced immersion
- **Keyboard Navigation**: Full keyboard support for accessibility
- **Performance Optimized**: Smooth 60fps rendering with efficient resource management

## 🛠️ Technologies Used

- **Three.js**: 3D graphics and WebGL rendering
- **Vanilla JavaScript**: Pure JS for optimal performance
- **CSS3**: Advanced styling with animations and effects
- **HTML5**: Semantic markup and modern web standards
- **Web Audio API**: Dynamic audio generation

## 🚀 Quick Start

### Option 1: Direct File Opening
Simply open `index.html` in a modern web browser.

### Option 2: Local Server (Recommended)
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## 🎮 Controls

- **Mouse**: Navigate and interact with 3D scene
- **Arrow Keys**: Navigate between sections
- **Audio Button**: Toggle atmospheric sound effects
- **Navigation Menu**: Click to jump to specific sections

## 🔧 Customization

### Personal Information
Update the following in `index.html`:
- Name and title
- Contact information
- Skills and technologies
- Project descriptions
- Personal statistics

### Visual Styling
Modify `styles.css` to adjust:
- Color schemes
- Typography
- Layout and spacing
- Animation effects

### 3D Elements
Customize `script.js` to change:
- 3D models and shapes
- Particle systems
- Camera movements
- Lighting effects
